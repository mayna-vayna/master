
9. Что из такое JSON и как с ним работать в Qt


Ответ:


JSON это текстовый формат обмена данными оснваный на JS, но 
не имеет с ним ни какой зависимости и может быть использован других 
языках програмирования. Чаще всего с ним столкнуться можно в REST API
при клиент серверной связи (но можно и с помощью XML) ,  в качестве  значений 
в JSON могут быть использованы: JSON-объекты, массивы, числа, литералы и строки 
JSON-объект — это неупорядоченное множество пар «ключ: значение».
Ключ — это название параметра, который мы передаем серверу. 
Он служит маркером для принимающей запрос системы
пример:
{

"query": "свое имя или придуманное !!!",

"count": 7

}

Для работы JSON Qt  предоставляет библиотеку QJson b  для того чтобы с ним работать,
 его нужно сначала получить и собрать.  QJson позволяет работать с JSON на основе QVariant 
что дает ему гибкость. Он умеет парсить, писать JSON и сиреализировать QObject
