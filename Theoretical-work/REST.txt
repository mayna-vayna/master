
8. Расскажите про сетевые REST запросы. Из чего состоит, какие виды
запросов бывают и чем они отличаются, что можно отправит


Ответ:


REST- обычный клиент-сервер запрос с использованием Http протокола. 
Клиентом в данном случае выступает наш браузер. Этот протокол описывает
взаимодействие между двумя компьютерами  построенное на базе сообщений, 
называемых "запрос Request" и " ответ Response ". Каждое сообщение состоит
 из трех частей: стартовая строка, заголовки и тело. При этом обязательной является 
только стартовая строка.Для разграничения действий с ресурсами на уровне HTTP-методов
 и были придуманы следующие варианты запросов: GET — получение ресурса, 
POST — создание ресурса, PUT — обновление ресурса, DELETE — удаление ресурса.
В качестве ресурсов(данных) может выступать Plain text — это обычный текст, 
XML — язык разметки информации,JSON — текстовый формат обмена данными,Binary.
